やつたこと

  • 勉学
    • 典型 90 を 1 (2) 問解いた．
      • 19 Pick Two
        • A_1 と同じ操作で取り除かれる数は A_2, A_4, A_6, ..., A_{2N} である．
          • A_1 と A_2 が同じ操作で取り除かれるなら，残つた列について同様にコストを計算して |A_1 - A_2| と足す．
          • A_1 と A_{2k} (k=2, ..., N) が同じ操作で取り除かれるなら，部分列 (A_2, ..., A_{2k-1}) と 部分列 (A_{2k+1}, ..., A_{2N}) についてコストを計算して， |A_1 - A_{2k}| と足す．
          • これらのうち最も小さいものが求めるコストである．
        • 愚直に書いても間に合はないかな，と思つたけどメモ化したら間に合つた．
          • Python だと TLE したけど C++ だと 50 倍以上高速化し AC できた．
          • 部分列 (A_h, ..., A_t) の値のメモを dict で c[f'{h:03}{t:03}'] で持つ代はりに長さ 401*401 のリストで c[h*401+t] で持つやうにしたら Python でも通つた． f-string が遅いのか？
