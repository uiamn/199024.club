        • 最初の提出でランダムケースだけ WA だつた．
          • 問題のサイズが大きくなると大体答へは 0 になる．
          • 各ビット毎に何通りの割当て方があるか求めてそれの総積を取るのだが，何故か 0 通りのときには 1 通りとして扱つてゐた．

      • 87 Chokudai's Demand: 考察類似行為をする．
        • X に対して f(X) を P スヌーク以下で到達可能であるやうな街の組の総数とすると，明らかに f は単調減少．よつてどうせにぶたんだらうと推測．
          • 上限はどうするんだ？と思つたが， X=P+1 としたときのある街間の P スヌーク以下で到達できる経路には，交通費が X スヌークの道路は含まれてゐるはずがないので，f(P+1) = f(P+2) = ... がわかる．なので，上限は P+1 にすればよい．
        • f(X) を求めるためにグラフに含まれる任意の 2 点間の距離を求めるアルゴリズムが必要だつたので，調べたら Floyd–Warshall 法を使へばよいことがわかる．
        • f(P+1) >= K+1 のときは，答へが 0 になることに注意する．
